
## วิธีการ Run Service API

```bash
cd .\json-server\
```

```bash
npm install
```

```bash
node index.js
```

ดูข้อมูล json ได้ที่ http://localhost:3000/

## วิธีการ Run Wab Appication

```bash
cd .\webapp\
```

```bash
npm install
```

```bash
npm run serve
```

Wab Appication จะ run อยู่ที่ http://localhost:8080/
